Setting Grid Columns

Everyone can set which columns they wish to see on the grid.

If you are the recipient of a Workspace that has been shared with Lock and Synch however (not the owner), your setting cannot be saved

You can drag-drop columns to arrange their order. Again however, if you cannot save the Workspace (not the owner), this ordering will not be preserved

Custom fields appear in bold.

The Project Template(s) of the projects you are viewing must have a field as Visible for you to see it in the column list
