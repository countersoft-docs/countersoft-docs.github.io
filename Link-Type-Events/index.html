Link Type Events

The following events are available before and after Link Type creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeLinkTypeListener

IssueLinkType BeforeLinkTypeCreated(LinkTypeEventArgs args);
IssueLinkType BeforeLinkTypeUpdated(LinkTypeEventArgs args);
IssueLinkType BeforeLinkTypeDeleted(LinkTypeEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterLinkTypeListener

void AfterLinkTypeCreated(LinkTypeEventArgs args);
void AfterLinkTypeUpdated(LinkTypeEventArgs args);
void AfterLinkTypeDeleted(LinkTypeEventArgs args);
