Project Label Events

The following events are available before and after Project Label creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeProjectLabelListener

ProjectLabel BeforeProjectLabelCreated(ProjectLabelEventArgs args);
ProjectLabel BeforeProjectLabelUpdated(ProjectLabelEventArgs args);
ProjectLabel BeforeProjectLabelDeleted(ProjectLabelEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterProjectLabelListener

void AfterProjectLabelCreated(ProjectLabelEventArgs args);
void AfterProjectLabelUpdated(ProjectLabelEventArgs args);
void AfterProjectLabelDeleted(ProjectLabelEventArgs args);
