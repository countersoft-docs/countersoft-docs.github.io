Permission Endpoint

Get All

This method returns a list of Permissions.

Get/api/admin/permissionsets

It returns a list of PermissionSetDto's.

Microsoft.NET example:

List<PermissionSetDto> permissions = serviceManager.Admin.GetPermissionSets();

JavaScript example:

var geminiUrl = "http://localhost/gemini/api/admin/permissionsets";
var geminiUsername = Base64.encode("manager:xvitjc5bmm"); // user : apikey

$.ajax({
    url: geminiUrl,
    type: "GET",
    headers: { "Authorization": "Basic " + geminiUsername },
    success: function (data) {
      alert('Success!');
    }
});
