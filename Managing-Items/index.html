Managing Items

An item can represent a task, ticket, change request, story, requirement, bug, feature request or enhancement or any other item type.

  • An item can be duplicated
  • An item can be moved between projects
  • An item can be assigned one or more resources
  • An item has associated workflow that depicts it's state
