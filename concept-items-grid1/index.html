Items Grid

Items within project can be displayed in a grid that can be sorted, filtered and ordered.

Every row in the grid represents an individual item.

  • You can filter instantly to find items of interest
  • You can export the items to Excel
  • You can perform bulk updates to make updates en masse
  • You can run Excel based reports on the items
