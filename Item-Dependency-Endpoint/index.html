Item Dependency Endpoint

Get

This method returns a list of Items.

Get/api/items/{itemid}/dependencies

The URL must have a Item Id. It returns a list of IssueDto's.

Microsoft.NET example:

List<IssueDto> items = serviceManager.Item.GetDependencies(36);

JavaScript example:

var geminiUrl = "http://localhost/gemini/api/items/36/dependencies";
var geminiUsername = Base64.encode("manager:xvitjc5bmm"); // user : apikey

$.ajax({
    url: geminiUrl,
    type: "GET",
    headers: { "Authorization": "Basic " + geminiUsername },
    success: function (data) {
      alert('Success!');
    }
});
