System Translations

For all active languages, you can manage the translation files from this screen.

Select the language for which you need to amend the translations. When the language is selected, the current translations will be shown.

Key
The translation key for the entry, useful to relate this item to the xml file.
English
This is the default translation in US English.
Translation
The translation (default) for the selected language.
Override
The current value used.

Click the Override value to edit or set the value to be used to override the pre-defined value.
