Version Events

The following events are available before and after Version creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeVersionListener

Version BeforeVersionCreated(VersionEventArgs args);
Version BeforeVersionUpdated(VersionEventArgs args);
Version BeforeVersionDeleted(VersionEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterVersionListener

void AfterVersionCreated(VersionEventArgs args);
void AfterVersionUpdated(VersionEventArgs args);
void AfterVersionDeleted(VersionEventArgs args);
