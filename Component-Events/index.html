Component Events

The following events are available before and after Component creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeComponentListener

Component BeforeComponentCreated(ComponentEventArgs args);
Component BeforeComponentUpdated(ComponentEventArgs args);
Component BeforeComponentDeleted(ComponentEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterComponentListener

void AfterComponentCreated(ComponentEventArgs args);
void AfterComponentUpdated(ComponentEventArgs args);
void AfterComponentDeleted(ComponentEventArgs args);
