Project Template Events

The following events are available before and after Project Template creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeProjectTemplateListener

InstalledProjectTemplate BeforeProjectTemplateCreated(ProjectTemplateEventArgs args);
InstalledProjectTemplate BeforeProjectTemplateUpdated(ProjectTemplateEventArgs args);
InstalledProjectTemplate BeforeProjectTemplateDeleted(ProjectTemplateEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterProjectTemplateListener

void AfterProjectTemplateCreated(ProjectTemplateEventArgs args);
void AfterProjectTemplateUpdated(ProjectTemplateEventArgs args);
void AfterProjectTemplateDeleted(ProjectTemplateEventArgs args);
