Updating Workspaces

A workspace is set to display the data as per the filter criteria. Nothing automatically updates the workspace structure, options or filters, you have to save your changes to it.

Click the save button to update the workspace with your changes.

If the workspace is shared with co-workers, they automatically receive the saved changes as well.
