Integrating Source Control

Integrations with the following Source Control systems are supported.

  • Git
  • Mercurial
  • GitHub
  • Bitbucket
  • Team Foundation Server 2010
  • Team Foundation Server 2012
  • Subversion (SVN)
  • Tortoise Subversion Client

The Saucery app provides Code Review capabilities for the following Source Control providers.

  • GitHub
  • Bitbucket
  • Subversion
  • Microsoft Team Foundation Server 2010
  • Microsoft Team Foundation Server 2012
