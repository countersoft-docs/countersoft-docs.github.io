Attaching Files

Every item can have multiple file attachments.

Click the add button to upload one or more files.

You can also drag-drop files onto the item itself to upload.

Hovering over an attachment allows you to delete it (subject to permissions).

Note If you do not see the Attachments section then you do not have permissions to upload files
