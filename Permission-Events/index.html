Permission Events

The following events are available before and after Permission creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforePermissionSetListener

PermissionSet BeforePermissionSetCreated(PermissionSetEventArgs args);
PermissionSet BeforePermissionSetUpdated(PermissionSetEventArgs args);
PermissionSet BeforePermissionSetDeleted(PermissionSetEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterPermissionSetListener

void AfterPermissionSetCreated(PermissionSetEventArgs args);
void AfterPermissionSetUpdated(PermissionSetEventArgs args);
void AfterPermissionSetDeleted(PermissionSetEventArgs args);
