System Languages

Gemini supports multiple languages, set at the user's profile level. Providing the correct translation file is provided, you can enable unlimited languages to suit your organizations needs

By default, the page is loaded with a filter set for the Active languages, but this can be reset by clearing the Active

Language
The language name
Code
The ISO language and culture code
Pre-installed
Whether the language is preinstalled as part of the Gemini installation
Active
Denotes if the language is available for use in Gemini

To activate a language, click on the Active column and the inline editing will appear. Check the box and it will be activated
