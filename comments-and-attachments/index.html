Comments and Attachments

Every item can have comments to aid co-worker and customer collaboration.

If you have enabled Visibility on your Project Termplate, you can control the visibility of an individual comment to ensure that only certain user groups can see it.

Important Gemini will not let one user amend another user's comments. You can delete comments if you have the permission to do so.

Every item can also have unlimited file attachments.
