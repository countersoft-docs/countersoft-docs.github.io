Importing Existing Data

Data from external sources can be imported using Gemini's Import Wizard.

NoteYou must be in a User Group with "Can Perform Bulk Operations" permission to use the Import Wizard.

NoteYour file should have column headings and the rows of data to be imported should NOT have gaps.

You can choose to import from Microsoft Excel and CSV file formats (PIPE, COMMA or TAB delimited).

You choose which fields to import and what they map to. Optionally, you can also ignore fields in the source file.

For every field value found in your import file you can map it to an existing value or add it during the import process.

Imported items will appear in the Items Grid.

Note The wizard can import multiple comments for one item by using the delimiter #!# but they have to be in one record with the item.

Note You can import dependencies by mapping a column to the Dependency field and entering either an existing item id or the row number in Rx format. R2 will be the item imported in the 2nd row of the file (1st data row when you have headers).

Note In some cases you might receive an error message about folder permssions. If this happens, please check that you have set folder permissions on the web server as specified in Gemini installation and upgrade instructions.
