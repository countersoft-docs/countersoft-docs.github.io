Resolution Events

The following events are available before and after Resolution creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeIssueResolutionListener

IssueResolution BeforeIssueResolutionCreated(IssueResolutionEventArgs args);
IssueResolution BeforeIssueResolutionUpdated(IssueResolutionEventArgs args);
IssueResolution BeforeIssueResolutionDeleted(IssueResolutionEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterIssueResolutionListener

void AfterIssueResolutionCreated(IssueResolutionEventArgs args);
void AfterIssueResolutionUpdated(IssueResolutionEventArgs args);
void AfterIssueResolutionDeleted(IssueResolutionEventArgs args);
