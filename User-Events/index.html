User Events

The following events are available before and after User creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeUserListener

User BeforeUserCreated(UserEventArgs args);
User BeforeUserUpdated(UserEventArgs args);
User BeforeUserDeleted(UserEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterUserListener

void AfterUserCreated(UserEventArgs args);
void AfterUserUpdated(UserEventArgs args);
void AfterUserDeleted(UserEventArgs args);
