Priority Events

The following events are available before and after Priority creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeIssuePriorityListener

IssuePriority BeforeIssuePriorityCreated(IssuePriorityEventArgs args);
IssuePriority BeforeIssuePriorityUpdated(IssuePriorityEventArgs args);
IssuePriority BeforeIssuePriorityDeleted(IssuePriorityEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterIssuePriorityListener

void AfterIssuePriorityCreated(IssuePriorityEventArgs args);
void AfterIssuePriorityUpdated(IssuePriorityEventArgs args);
void AfterIssuePriorityDeleted(IssuePriorityEventArgs args);
