Project Changelog Endpoint

Get

This method returns a list of items for a Version.

Get/api/items/{versionid}/changelog

The URL must have a Version Id. The return type is a list of items.

Microsoft.NET example:

ItemService itemManager = new ItemService();
var items = itemManager.GetChangelog(18);

JavaScript example:

var geminiUrl = "http://localhost/gemini/api/items/18/changelog";
var geminiUsername = Base64.encode("manager:xvitjc5bmm"); // user : apikey

$.ajax({
    url: geminiUrl,
    type: "GET",
    headers: { "Authorization": "Basic " + geminiUsername },
    success: function (data) {
        alert('Success!');
    }
});
