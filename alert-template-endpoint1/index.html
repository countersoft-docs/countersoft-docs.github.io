Alert Template Endpoint

Get All

This method returns a list with all the Alert Templates.

Get/api/admin/alerts/templates

It returns a list of AlertTemplate Entities.

Microsoft.NET example:

List<AlertTemplate> alertTemplates = serviceManager.Admin.GetAlertTemplates();

JavaScript example:

var geminiUrl = "http://localhost/gemini/api/admin/alerts/templates";
var geminiUsername = Base64.encode("manager:xvitjc5bmm"); // user : apikey

$.ajax({
    url: geminiUrl,
    type: "GET",
    headers: { "Authorization": "Basic " + geminiUsername },
    success: function (data) {
      alert('Success!');
    }
});
