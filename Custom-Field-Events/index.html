Custom Field Events

The following events are available before and after Custom Field creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeCustomFieldListener

CustomField BeforeCustomFieldCreated(CustomFieldEventArgs args);
CustomField BeforeCustomFieldUpdated(CustomFieldEventArgs args);
CustomField BeforeCustomFieldDeleted(CustomFieldEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterCustomFieldListener

void AfterCustomFieldCreated(CustomFieldEventArgs args);
void AfterCustomFieldUpdated(CustomFieldEventArgs args);
void AfterCustomFieldDeleted(CustomFieldEventArgs args);
