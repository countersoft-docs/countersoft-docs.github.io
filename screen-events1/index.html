Screen Events

The following events are available before and after Screen creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeScreenListener

Screen BeforeScreenCreated(ScreenEventArgs args);
Screen BeforeScreenUpdated(ScreenEventArgs args);
Screen BeforeScreenDeleted(ScreenEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterScreenListener

void AfterScreenCreated(ScreenEventArgs args);
void AfterScreenUpdated(ScreenEventArgs args);
void AfterScreenDeleted(ScreenEventArgs args);
