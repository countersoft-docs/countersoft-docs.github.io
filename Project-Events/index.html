Project Events

The following events are available before and after Project creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeProjectListener

Project BeforeProjectCreated(ProjectEventArgs args);
Project BeforeProjectUpdated(ProjectEventArgs args);
Project BeforeProjectDeleted(ProjectEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterProjectListener

void AfterProjectCreated(ProjectEventArgs args);
void AfterProjectUpdated(ProjectEventArgs args);
void AfterProjectDeleted(ProjectEventArgs args);
