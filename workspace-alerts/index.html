Workspace Alerts

You can choose to receive email alerts whenever something changes in the workspace

By default, one summary email is sent detailing all changes. However, you can opt to receive individual email alerts for every change. This is an option on the User Profile

If you subscribe co-workers to your alerts, they can un-subscribe at any time. This is also an option on the User Profile - Alerts tab
