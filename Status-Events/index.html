Status Events

The following events are available before and after Status creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeIssueStatusListener

IssueStatus BeforeIssueStatusCreated(IssueStatusEventArgs args);
IssueStatus BeforeIssueStatusUpdated(IssueStatusEventArgs args);
IssueStatus BeforeIssueStatusDeleted(IssueStatusEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterIssueStatusListener

void AfterIssueStatusCreated(IssueStatusEventArgs args);
void AfterIssueStatusUpdated(IssueStatusEventArgs args);
void AfterIssueStatusDeleted(IssueStatusEventArgs args);
