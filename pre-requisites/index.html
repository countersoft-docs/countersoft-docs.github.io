System Requirements

The following are the recommended minimum system requirements to run Gemini. For installations containing larger amounts of users, increase your system resources accordingly.

Minimum System Requirements

  • Quad Core CPU 2.0Ghz (or equivalent)
  • 8GB RAM minimum
  • 10GB HDD of FREE space at 15,000 RPM
  • Windows 2008 R2 OS or above
  • SQL Server 2008 R2 or above - some features of Gemini, such as As Of queries, are not available on versions of SQL Server prior to SQL Server 2016
    • SQL Express editions are not recommended for production, although Gemini will work with them.

.Net Framework For Gemini 6.8.1 and above .Net 4.5.2 or above is required. Earlier versions require .Net 4.0.x

.Net Framework For Gemini 6.9.2 and above .Net 4.7.2 is the minimum required version.
