Project Document Events

The following events are available before and after Project Document creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeProjectDocumentListener

ProjectDocument BeforeProjectDocumentCreated(ProjectDocumentEventArgs args);
ProjectDocument BeforeProjectDocumentUpdated(ProjectDocumentEventArgs args);
ProjectDocument BeforeProjectDocumentDeleted(ProjectDocumentEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterProjectDocumentListener

void AfterProjectDocumentCreated(ProjectDocumentEventArgs args);
void AfterProjectDocumentUpdated(ProjectDocumentEventArgs args);
void AfterProjectDocumentDeleted(ProjectDocumentEventArgs args);
