Time Logs Events

The following events are available before and after Time Logs creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeIssueTimeTrackingListener

IssueTimeTracking BeforeIssueTimeTrackingCreated(IssueTimeTrackingEventArgs args);
IssueTimeTracking BeforeIssueTimeTrackingUpdated(IssueTimeTrackingEventArgs args);
IssueTimeTracking BeforeIssueTimeTrackingDeleted(IssueTimeTrackingEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterIssueTimeTrackingListener

void AfterIssueTimeTrackingCreated(IssueTimeTrackingEventArgs args);
void AfterIssueTimeTrackingUpdated(IssueTimeTrackingEventArgs args);
void AfterIssueTimeTrackingDeleted(IssueTimeTrackingEventArgs args);
