Navigation Card Events

Navigation Card is the internal name for workspaces in Gemini.

The following events are available before and after Navigation Card creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeNavigationCardListener

NavigationCard BeforeNavigationCardCreated(NavigationCardEventArgs args);
NavigationCard BeforeNavigationCardUpdated(NavigationCardEventArgs args);
NavigationCard BeforeNavigationCardDeleted(NavigationCardEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterNavigationCardListener

void AfterNavigationCardCreated(NavigationCardEventArgs args);
void AfterNavigationCardUpdated(NavigationCardEventArgs args);
void AfterNavigationCardDeleted(NavigationCardEventArgs args);
