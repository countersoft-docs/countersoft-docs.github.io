Item Repeat Endpoint

Get

This method returns a list of Items which are repeated.

Get/api/items/repeated

It returns a list of IssueDto's.

Microsoft.NET example:

List<IssueDto> items = serviceManager.Item.GetRepeated();

JavaScript example:

var geminiUrl = "http://localhost/gemini/api/items/repeated";
var geminiUsername = Base64.encode("manager:xvitjc5bmm"); // user : apikey

$.ajax({
    url: geminiUrl,
    type: "GET",
    headers: { "Authorization": "Basic " + geminiUsername },
    success: function (data) {
      alert('Success!');
    }
});
