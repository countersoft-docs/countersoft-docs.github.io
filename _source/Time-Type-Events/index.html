Time Type Events

The following events are available before and after Time Type creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeTimeTypeListener

TimeType BeforeTimeTypeCreated(TimeTypeEventArgs args);
TimeType BeforeTimeTypeUpdated(TimeTypeEventArgs args);
TimeType BeforeTimeTypeDeleted(TimeTypeEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterTimeTypeListener

void AfterTimeTypeCreated(TimeTypeEventArgs args);
void AfterTimeTypeUpdated(TimeTypeEventArgs args);
void AfterTimeTypeDeleted(TimeTypeEventArgs args);
