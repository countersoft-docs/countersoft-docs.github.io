Process Type Events

The following events are available before and after Process Type creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeIssueTypeListener

IssueType BeforeIssueTypeCreated(IssueTypeEventArgs args);
IssueType BeforeIssueTypeUpdated(IssueTypeEventArgs args);
IssueType BeforeIssueTypeDeleted(IssueTypeEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterIssueTypeListener

void AfterIssueTypeCreated(IssueTypeEventArgs args);
void AfterIssueTypeUpdated(IssueTypeEventArgs args);
void AfterIssueTypeDeleted(IssueTypeEventArgs args);
