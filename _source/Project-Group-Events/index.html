Project Group Events

The following events are available before and after Project Group creation, update and deletion.

Before

Use these events to modify the entity before it is persisted.

IBeforeProjectGroupListener

ProjectGroup BeforeProjectGroupCreated(ProjectGroupEventArgs args);
ProjectGroup BeforeProjectGroupUpdated(ProjectGroupEventArgs args);
ProjectGroup BeforeProjectGroupDeleted(ProjectGroupEventArgs args);

After

Use these events to be notified after an entity operation.

IAfterProjectGroupListener

void AfterProjectGroupCreated(ProjectGroupEventArgs args);
void AfterProjectGroupUpdated(ProjectGroupEventArgs args);
void AfterProjectGroupDeleted(ProjectGroupEventArgs args);
