Developer Guide

Developers can change and extend Gemini using the extensibility points.

A RESTful API provides complete access to user data and all meta-data.

Listen to events within Gemini and interact as required.

Create and deploy custom apps the extend the user interface.

Skin the application to match corporate branding.
